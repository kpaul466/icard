# iCard Generator 🪪

The **iCard Generator** is a fast and efficient web-based tool designed to create professional identity cards. This application allows users to input personal details, upload images, and generate a standardized ID card format directly in the browser.

**Live Demo:** [https://kpaul466.github.io/icard/](https://kpaul466.github.io/icard/)

---

## 🚀 Getting Started

The interface is divided into two primary sections:
1.  **Input Form:** Where you enter the individual's data.
2.  **Live Preview:** Where the ID card is rendered in real-time.

---

## 🛠 How to Use

Follow these steps to generate a professional card:

### 1. Enter Personal Information
Fill in the text fields in the form. Standard fields include:
* **Full Name:** The name as it should appear on the card.
* **Designation:** The official role or job title.
* **ID/Employee Number:** Unique identifier for the individual.
* **Contact Info:** Phone number or email address.
* **Department/Office:** The specific branch or team.

### 2. Upload a Photo
* Click the **Upload** or **Choose File** button.
* Select a portrait-style photo from your device.
* **Pro-Tip:** For the best professional look, use a high-resolution image with a solid, neutral background.

### 3. Review the Layout
The **Live Preview** updates as you type. 
* Check for spelling accuracy.
* Ensure the photo is correctly framed within the designated area.
* Verify that long text fits within the card boundaries.

---

## 📥 Exporting and Printing

Once the preview looks correct, you can save or print your card:

| Action | Description |
| :--- | :--- |
| **Download** | Click the 'Download' or 'Generate' button to save the card as an image or PDF. |
| **Print** | Use `Ctrl + P` (Windows) or `Cmd + P` (Mac) to open the print dialog. |
| **Paper Size** | For physical cards, set the paper size to **CR80** (Standard ID size). |

---

## 💡 Troubleshooting & Tips

* **Image Issues:** Ensure the image is in `.jpg` or `.png` format and under 2MB.
* **Text Overflow:** If a name is too long, consider using initials to maintain the layout.
* **Browser Support:** Best experienced on **Google Chrome**, **Mozilla Firefox**, or **Microsoft Edge**.

---

## 🔒 Privacy & Security

The **iCard Generator** is a client-side application hosted on GitHub Pages. 
* **No Data Storage:** All processing happens locally in your browser. 
* **Privacy:** Your personal information and uploaded photos are **never** sent to a server or stored externally.
